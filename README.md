cmc-json
========

This is a PHP script that uses simple HTML DOM parsing to get the top 100 coins from coinmarketcap.com. 
Then, the script converts Coinmarketcap.com into JSON data.

Possible usage: 
```
$ php market_cap.php > coins.json
```
